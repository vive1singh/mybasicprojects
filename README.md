# Chitbowlgame
Interactive chit‑picking party game in the browser with a glass‑bowl UI, pick‑and‑reveal chits, team scoreboard, history log, and localStorage‑saved sessions (HTML, CSS, JavaScript).

Chit Bowl Game is a browser‑based party game where players add teams and custom chits, then “draw” from a glass‑style bowl. Each draw is a suspenseful two‑step flow (Pick → Reveal) that updates team scores, highlights the leading team, and keeps a full history of picks. The app is built with plain HTML, CSS, and JavaScript, and saves game state in localStorage so you can refresh without losing progress.
