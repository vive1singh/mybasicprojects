🎮 Party Game Collection (Browser Based)
Browser-based party games built with HTML, CSS, and JavaScript: an interactive chit-picking game with a glass-bowl UI, and a classic Ludo-style board game implementation.

🧩 Project 1 – Chit-Picking Party Game
Interactive chit-picking party game for groups, with a virtual glass bowl, team scoreboard, history log, and auto-saved sessions using localStorage.

Features
Glass-bowl style interface to pick and reveal chits with animations.
​

Multiple teams with a live scoreboard and manual score controls.
​

History panel showing which chit was picked by which team and when.
​

Game state (teams, scores, remaining chits, history) saved in localStorage.
​

Runs completely in the browser – no backend required.
​

How to Run
Clone this repository and open it in your editor.
​

Open chitgame.html directly in your browser, or use a Live Server extension.
​

Add teams, start picking chits, and update scores as you play.
​

🎲 Project 2 – Ludo Game (Project Ludo)
A browser-based Ludo-style board game that simulates the classic tabletop experience with tokens, dice rolls, and turn-based gameplay.

Features
HTML/CSS powered Ludo board and tokens.
​

JavaScript-controlled dice logic and turn order.
​

Token movement rules inspired by classic Ludo (safe cells, home paths, etc.).
​

Designed for quick local multiplayer on a single device.
​

How to Run
In the same repo, open projectludo.html in your browser.
​

Use the UI to roll the dice and move tokens according to the prompts.
​
